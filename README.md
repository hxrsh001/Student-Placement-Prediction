# Student Placement Prediction

## Overview

This project predicts whether a student is likely to be placed based on academic and performance-related features using Machine Learning.

## Features

- Label Encoding
- Feature Scaling
- Train-Test Split
- Linear Regression Model
- Model Evaluation
- Student Placement Prediction
- Data Visualization using Seaborn
- Model Saving using Joblib

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Seaborn
- Matplotlib
- Joblib

## Dataset Features

- Study Hours
- Attendance
- Sleep Hours
- Internet Usage
- Assignments Completed
- Previous Score
- Placement Status

## Output

The project predicts whether a student is likely to be **Placed** or **Not Placed** based on the given input features.
